Big City Life API

Get your API key here!

All the available end-points...

For a user -

GET a user using the unique API key

("http://localhost:8080/api/user/*APIKEY*")

PATCH a user details the unique API key

("http://localhost:8080/api/user/*APIKEY*")

DELETE a user using the unique API key

("http://localhost:8080/api/user/*APIKEY*")

For a city -

Get all cities currently in the API

("http://localhost:8080/api/city/*APIKEY*")

GET all cities currently in the API that are owned by your unique API key

("http://localhost:8080/api/city/*APIKEY*/my-cities")

POST a city to the API

("http://localhost:8080/api/user/*APIKEY*")

DELETE a city using the city name (Your API key must be the owner of the POST)

("http://localhost:8080/api/user/*APIKEY*/*CITYNAME*")

PATCH a city using the city name (Your API key must be the owner of the POST)

("http://localhost:8080/api/user/*APIKEY*/*CITYNAME*")

GET a single city using the city name

("http://localhost:8080/api/city/*APIKEY*/*CITYNAME*")

GET the number of cities currently in the API

("http://localhost:8080/api/city/*APIKEY*/number-of-cities")

GET the number of countries currently in the API

("http://localhost:8080/api/city/*APIKEY*/number-of-countries")

GET a list of all cpuntries currently in the API

("http://localhost:8080/api/city/*APIKEY*/list-of-countries")

GET the top 10 most populace cities currently in the API

("http://localhost:8080/api/city/*APIKEY*/most-populace-cities")

GET the top 10 least populace cities currently in the API

("http://localhost:8080/api/city/*APIKEY*/least-populace-cities")

GET a random city from the API

("http://localhost:8080/api/city/*APIKEY*/random-city")

For a review -

GET all reviews connected to your unique API key

("http://localhost:8080/api/city/*APIKEY*/my-reviews")

POST a review for a city in the API

("http://localhost:8080/api/city/*APIKEY*/submit-review")

PATCH a review connected to your unique API key

("http://localhost:8080/api/city/*APIKEY*/*REVIEWID")

DELETE all reviews connected to your unique API key

("http://localhost:8080/api/city/*APIKEY*/my-reviews")

DELETE a single review connected to your unique API key

("http://localhost:8080/api/city/*APIKEY*/*REVIEWID*")

GET all reviews for a city by rating (1 - 5)

("http://localhost:8080/api/city/*APIKEY*/*CITYNAME*/*RATING*")
