All the available end-points...

For a user -

POST a new user to get a unique API key

("http://localhost:8080/api/user/*APIKEY*/deactivate-apikey")

Enter API-key to the URL. Enter firstName, lastName and email to the request body

GET a user using the unique API key

("http://localhost:8080/api/user/*APIKEY*")

Enter API-key to the URL to retrieve and show all the users currently in the database.

PATCH a user details the unique API key

("http://localhost:8080/api/user/*APIKEY*")

Enter API-key and the values you want to change to the request body to update the user.

PATCH a user to deactivate the API key

("http://localhost:8080/api/user/*APIKEY*/deactivate-apikey")

Enter API-key to the URL to deactivate it. This will not remove content from the API connected to the API key.

For a city -

GET all cities currently in the API

("http://localhost:8080/api/city/*APIKEY*")

Enter API-key to the URL to retrieve and show all the cities currently in the database.

GET all cities currently in the API that are owned by your unique API key

("http://localhost:8080/api/city/*APIKEY*/my-cities")

Enter API-key to the URL to show your created cities

POST a city to the API

("http://localhost:8080/api/city/*APIKEY*")

Enter API-key to the URL. Enter cityName, country, description, population and imageUrl(String) to the request body (JSON). Population values: Min value = 1,000, Max value = 100,000,000

DELETE a city using the city name (Your API key must be the owner of the POST)

("http://localhost:8080/api/city/*APIKEY*/*CITYNAME*")

Enter cityId & the city creator API-key to the URL to delete a city

PATCH a city using the city name (Your API key must be the owner of the POST)

("http://localhost:8080/api/city/*APIKEY*/*CITYNAME*")

Enter API-key to the URL. Enter cityName, country, description, imageUrl and population to the request body(JSON)

GET a single city using the city name

("http://localhost:8080/api/city/*APIKEY*/*CITYNAME*")

Enter API-key and cityName to the URL to show the city

GET the number of cities currently in the API

("http://localhost:8080/api/city/*APIKEY*/number-of-cities")

Enter API-key to the URL to show how many cities that are in the database

GET the number of countries currently in the API

("http://localhost:8080/api/city/*APIKEY*/number-of-countries")

Enter API-key to the URL to show how many countries that are in the database

GET a list of all countries currently in the API

("http://localhost:8080/api/city/*APIKEY*/list-of-countries")

Enter API-key to the URL to show a list off all countries

GET the top 10 most populace cities currently in the API

("http://localhost:8080/api/city/*APIKEY*/most-populace-cities")

Enter API-key to the URL to show the ten most populated cities

GET the top 10 least populace cities currently in the API

("http://localhost:8080/api/city/*APIKEY*/least-populace-cities")

Enter API-key to the URL to show the ten least populated cities

GET a random city from the API

("http://localhost:8080/api/city/*APIKEY*/random-city")

Enter API-key to the URL to show a random city

For a review -

GET all reviews connected to your unique API key

("http://localhost:8080/api/city/*APIKEY*/my-reviews")

Enter API-key to the URL to retrieve and show all the reviews currently in the database connected to a specific api key.

POST a review for a city in the API

("http://localhost:8080/api/city/*APIKEY*/submit-review")

Enter API-key to the URL and submit a review of the city as a JSON object. The Review object needs to contain a review, rating and a cityId(can be seen in the city object)

PATCH a review connected to your unique API key

("http://localhost:8080/api/city/*APIKEY*/*REVIEWID")

Enter the review creators API-key to the URL. Enter reviewId, review and rating to the request body (JSON)

DELETE all reviews connected to your unique API key

("http://localhost:8080/api/city/*APIKEY*/my-reviews")

Enter API-key to the URL to delete all reviews associated to that API key

DELETE a single review connected to your unique API key

("http://localhost:8080/api/city/*APIKEY*/*REVIEWID*")

Enter the review creators API-key and the reviewId to the URL to delete the review

GET all reviews for a city by rating (1 - 5)

("http://localhost:8080/api/city/*APIKEY*/*CITYNAME*/*RATING*")

Enter API-key and a rating to the URL to show all reviews of that city with that rating
